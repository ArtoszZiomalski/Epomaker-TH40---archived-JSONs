# Epomaker-TH40---archived-JSONs
This repository contains archived JSON files for EPOMAKER TH40 keyboard. If you have trouble configuring your keyboard with VIA when using JSON from EPOMAKER's website, you can try importing this one instead.
### Is this file legit?
Yes, this file was obtained by using Wayback Machine and EPOMAKER's older link to JSON files. The link was obtained from user manual.
