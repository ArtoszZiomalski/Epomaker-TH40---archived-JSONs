# Epomaker-TH40---archived-JSONs
This repository contains archived JSON files for EPOMAKER TH40 keyboard. If you have trouble configuring your keyboard with VIA when using JSON from EPOMAKER's website, you can try importing these files instead.
### Which file should I choose?
USB when your keyboard is set to USB wired mode. If your keyboard is connected using a 2.4Ghz dongle, then use the other file.
### Are those files legit?
Yes, those files were obtained by using Wayback Machine and EPOMAKER's older links to JSON files. Links were obtained from user manual.
